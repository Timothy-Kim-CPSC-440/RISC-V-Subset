# RISC-V-Subset

## Running the Program
- Open the Terminal application
- Navigate to the directory which contains main.py
- Enter into the terminal "python main.py" or "python3 main.py"

## Running Unit Tests
- Open the Terminal application
- Navigate to the directory which contains unittests.py
- Enter into the terminal "python unittests.py" or "python3 unittests.py"
